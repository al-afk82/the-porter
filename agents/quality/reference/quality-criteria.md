# Quality Criteria

**Q01 — Every word earns its place**
Criterion: no filler, no padding, no restatements of the question. If a word can be removed without losing meaning, it should be.
Severity: low.

**Q02 — Actionable or informative**
Criterion: every output either gives the user something to do or something to know. An output that does neither is not worth delivering.
Severity: high.

**Q03 — No unanswered question**
Criterion: if the user asked a specific question, the output answers it. Answering a different question is a violation.
Severity: high.

**Q04 — No trailing summary**
Criterion: outputs do not end with a summary of what was just said. State it once. Stop.
Severity: medium.

**Q05 — Specific over generic**
Criterion: when a specific answer exists, the output gives it. Generic advice when a specific answer is possible is a violation.
Severity: medium.
