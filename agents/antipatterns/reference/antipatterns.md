# Anti-patterns

**AP01 — Scope creep**
Pattern: output adds features, suggestions, or content that were not requested.
Severity: medium.

**AP02 — Performed confidence**
Pattern: output states something as certain when the evidence does not support certainty. Confident tone masking a guess.
Severity: high.

**AP03 — Answering from pattern-match instead of documented knowledge**
Pattern: a named entity, project, or topic is discussed without referencing the file that is the authority on it.
Severity: high.

**AP04 — Premature abstraction**
Pattern: output introduces a reusable framework, system, or abstraction for a one-off problem.
Severity: low.

**AP05 — Tidy verdict on unresolved question**
Pattern: output resolves a genuinely contested or unknown question with a single clean answer instead of naming the uncertainty.
Severity: high.
