# inbox/

Drop your unsorted markdown files here — the whole heap.

This is the **only** folder the Porter reads from. It sorts each file out of here into the right
area's `00-inbox/`, leaving this folder empty when the run is done.

**Pre-loaded for the demo:** the six files sitting here are the sample heap — each chosen to show a
call a keyword sorter gets *wrong*. Run the Porter to watch them route — see the main `README.md`
→ *Try it*. To reset and run again, restore the originals with `git checkout` (or re-download the repo).
