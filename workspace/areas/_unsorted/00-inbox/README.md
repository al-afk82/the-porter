# _unsorted / 00-inbox/

The catch-all. When **no clear area fits**, the Porter routes the file here and **proposes a new area** (a name + a one-line "what belongs here") in the file's stamp.

This is the pressure valve — it lets the Porter stay fast and honest: nothing gets forced into a wrong pile, nothing gets lost. When the same new area keeps getting proposed, add it to `reference/destinations.md` and re-run.
